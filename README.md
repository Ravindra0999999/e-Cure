A full-stack e-commerce healthcare platform for browsing medicines, managing carts,
placing orders, and handling user authentication.

## Tech Stack
- Frontend: React, TypeScript, Vite, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB
- State Management: Redux Toolkit

## Features
- User authentication (register/login)
- Browse medicines & stores
- Cart & order management
- Reviews and ratings
- Protected routes

## Run Locally

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev
